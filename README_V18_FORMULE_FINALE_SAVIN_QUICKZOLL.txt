V18 finale : formule SAVIN' + QuickZoll validée.

Pour une ligne en EUR, le coût réel final CHF est calculé ainsi :

((EUR TTC × (1 - Savin')) × taux EUR/CHF)
+
((EUR TTC / 1.20) × taux EUR/CHF × TVA CH)

Ce qui correspond à :
1. Prix TTC payé en Europe.
2. Remboursement SAVIN' calculé sur le TTC.
3. Conversion du montant net après SAVIN' en CHF.
4. TVA suisse QuickZoll calculée sur la valeur HT étrangère.
5. Ajout de cette TVA au coût réel.

Cette formule est utilisée partout :
- coût net CHF affiché ;
- dépenses prévues ;
- solde restant ;
- dépenses du mois ;
- solde cumulé ;
- solde après achat ;
- export CSV.

Aucun SQL Supabase à lancer.
